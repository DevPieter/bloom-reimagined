# Bloom Reimagined - Spicetify Theme

**Bloom Reimagined** is a [Spicetify](https://spicetify.app/) theme inspired by [Bloom](https://github.com/nimsandu/spicetify-bloom).

> [!IMPORTANT]
> This is my first Spicetify theme, and I'm not a designer. I just wanted to experiment with Spicetify and see what I could come up with.

## Motivation

Bloom is an amazing theme, but unfortunately, it hasn't been updated in a while and is partially broken. I was already thinking about creating my own theme, so I decided to use Bloom as inspiration and create my own version.
